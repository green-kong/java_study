### Java에서 사용되는 용어들

- **Java SE** : Java Platform, Standart Edition
  Java의 표준안, js의 es와 같은 비슷한 의미임, JCP라는 조직을 통해 만들어진다.

- **JDK**(Java Development Kit) : Java SE 표준안에 따라 만들어진 소프트 웨어.
  자발 개발과 컴파일을 위한 각종 도구가 들어있다. (for dev)

- **JRE**(Java Runtime Environment) :
  Java로 만들어진 프로그램이 구동 되기 위한 환경 JDK안에 포함되어 있다. (for client)

- **JVM**(Java Virtual Machine) :
  Java가 실제로 구동하는 환경. 하드웨어나 운영체제에 따라 달라질 수 있는 호환성의 문제를 JVM이 알아서 해결한다.
  즉, 어떤 환경에서도 동일하게 실행되게끔 만들어 주는 것이 JVM의 역할이다.
